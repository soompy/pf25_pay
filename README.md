### Atomic Design 기반 컴포넌트 라이브러리

#### 📦 Atoms (기본 컴포넌트)
```tsx
import { Button, Input, Badge, Avatar } from '@/components/ui';

// 다양한 variant와 size 지원
<Button variant="primary" size="lg">전송하기</Button>
<Input leftIcon={<Mail />} placeholder="이메일 주소" />
<Badge variant="success" dot>온라인</Badge>
```

#### 🧬 Molecules (조합 컴포넌트)  
```tsx
import { Card, Modal, FormField, SearchBox } from '@/components/ui';

// 복잡한 UI 패턴을 간단하게
<Card padding="lg" hoverable>
  <FormField label="금액" error={errors.amount}>
    <Input type="number" {...register('amount')} />
  </FormField>
</Card>
```

#### 🦠 Organisms (복잡한 컴포넌트)
```tsx
import { DataTable, PaymentForm, ContactList } from '@/components/ui';

// 비즈니스 로직이 포함된 고급 컴포넌트

  columns={columns}

/>
```

## 🎭 CSS 커스터마이징

### 6가지 커스터마이징 방법

#### 1. **기본 Props 활용**
```tsx
<Button variant="outline" size="lg" className="shadow-xl">
  Props로 커스터마이징
</Button>
```

#### 2. **CSS Variables 기반 테마**
```tsx
<Button 
  theme="custom"
  style={{
    '--color-primary-500': '#ff6b6b',
    '--color-primary-600': '#ee5a52',
  }}
>
  동적 컬러 변경
</Button>
```

#### 3. **useStyleOverride Hook**
```tsx
const { className, style } = useStyleOverride({
  baseClasses: 'rounded-xl p-4',
  conditionalClasses: {
    'bg-gradient-to-r from-blue-500 to-purple-500': true,
    'hover:scale-105': isInteractive,
  },
  customClasses: 'animate-pulse-soft',
});
```

#### 4. **테마 스위처**
```tsx
import { ThemeSwitcher } from '@/components/ui';

// 실시간 테마 변경
<ThemeSwitcher showCustomizer variant="dropdown" />
```

#### 5. **Tailwind 확장 클래스**
```tsx
<div className="btn-primary card glass gradient-border">
  확장된 유틸리티 클래스
</div>
```

#### 6. **전역 테마 시스템**
```tsx
// 라이트/다크 모드 + 컬러 테마
<ThemeProvider defaultTheme="dark">
  <App />
</ThemeProvider>
```

### 지원하는 테마

- **라이트/다크 모드**: 시스템 설정 연동
- **컬러 테마**: Blue, Green, Purple, Orange, Pink
- **커스텀 테마**: 실시간 컬러 피커로 생성
- **프리셋**: 인기 있는 색상 조합 제공
